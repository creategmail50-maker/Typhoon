# Typhoon# 🌪️ 颱風巴威最新動態

這是一個使用 GitHub Pages 部署的靜態網頁，提供颱風巴威的最新路徑與影響資訊。

## 部署方式
1. 建立 Repository 並上傳 `index.html`
2. 在 Repository 設定中啟用 **GitHub Pages**
   - Source: `main` branch
   - Folder: `/ (root)`
3. 完成後即可透過 `https://你的帳號.github.io/typhoon-bavi/` 瀏覽

## 自動部署
已附上 GitHub Actions 設定檔，push 後會自動更新 Pages。
